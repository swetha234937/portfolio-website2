# Swetha M | Cloud & DevOps Portfolio

 A modern personal portfolio website built using HTML, CSS, and JavaScript, showcasing my skills, projects, and experience in Cloud Computing and DevOps.

---

## Features

- Responsive design (mobile-friendly)
-  Dark/Light mode toggle
-  Smooth scrolling navigation
-  Scroll animations
-  Projects showcase section
-  Resume download option
-  Contact form (UI)

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript
- AWS S3 (for hosting)
- Terraform (for infrastructure automation)
